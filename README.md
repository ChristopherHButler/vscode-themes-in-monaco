# Sandbox
---
This is a minimal, vanilla react sandbox (does not use CRA) setup for writing modern (ES6+) jsx code.

## Features

Comes with a few convenient features:

 - Bundling (webpack)
 - Transpiling (babel)
 - Hot reloading (webpack)
 - Test watching (jest)

## Getting Started

fire it up with

```
> npm run start
```
